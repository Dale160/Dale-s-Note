# 7.4 创建多线程的方式二： 实现 Runnable接口
1. 创建一个实现了Runnable接口的类
2. 实现类，去实现Runnable 接口中的抽象方法： run();
3. 创建实现类的对象
4. 将此对象 作为参数传递到Thread类的构造器中，创建Thread类的对象
5. 通过Thread类的对象，调用stat()的方法。

# 实例代码：
```java
package Test;

/**
 * 创建多线程的方式二： 实现
 *
 * 1. 创建一个实现了Runnable接口的类
 * 2. 实现类，去实现Runnable 接口中的抽象方法： run();
 * 3. 创建实现类的对象
 * 4. 将此 实现类的对象 作为参数传递到Thread类的构造器中，创建Thread类的对象
 * 5. 通过Thread类的对象，调用stat()的方法。
 *
 * @author Dale's File
 * @creat 2021-03-14-16:09
 */

// 1. 创建一个实现了Runnable接口的类
class MThread implements Runnable{

//2. 实现类，去实现Runnable 接口中的抽象方法： run();
    @Override
    public void run() {
        for (int i = 0; i < 100; i++) {
            if (i % 2 == 0){
                System.out.println(Thread.currentThread().getName() + ":" + i);
            }

        }
    }
}


public class ThreadTest2 {
    public static void main(String[] args) {
        //3. 创建实现类的对象
        MThread mThread = new MThread();

//        4. 将此 实现类的对象 作为参数传递到Thread类的构造器中，创建Thread类的对象
        Thread t1 = new Thread(mThread);
        t1.setName("线程1");

//        5. 通过Thread类的对象，调用stat()的方法。 ①启动线程
//        ②调用当前线程的run() 方法 --->调用了Runnable 类型的target的run()的方法

        t1.start();

        //再启动一个线程，遍历100以内的偶数
        Thread t2 = new Thread(mThread);
        t2.setName("线程2");
        t2.start();
    }
}
```

package Test;

/**
 *
 * 例子： 创建三个窗口卖票，票的总数为100张 尚硅谷423，使用Runnable的方式
 *
 * @author Dale's File
 * @creat 2021-03-14-18:36
 */

class Windows implements Runnable{

    private static int ticket = 100;

    @Override
    public void run() {
        while (true) {
            if (ticket > 0) {
                    System.out.println(Thread.currentThread().getName() + ":" + ticket);
                    ticket--;

                }else
                    break;
            }

    }
}

public class WindowTest2 {
    public void main(String[] args) {
        Windows w1 = new Windows();

        Thread t1 = new Thread(w1);
        t1.setName("第一个窗口");
        t1.start();

        Thread t2 = new Thread(w1);
        t2.setName("第2个窗口");
        t2.start();

        Thread t3 = new Thread(w1);
        t2.setName("第3个窗口");
        t2.start();

    }
}
