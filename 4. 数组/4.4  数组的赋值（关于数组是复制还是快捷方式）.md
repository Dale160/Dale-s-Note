# 5.3  数组的赋值（关于数组是复制还是快捷方式）

## 例题：以及数值的完全复制，还是地址的复制
```java
/*
* 3.数组的赋值、翻转、**查找**（线性查找、**二分法查找**）尚硅谷160
*
* 思考array1和array2是什么关系？array1和array2的地址值相同，都指向了堆空间的唯一一个实体。
*
* 拓展：修改题目，实现array2对array1的复制
* */

package com.atguigu.exer;

public class ArrayExer2 {
    public static void main(String[] args) {
        int [] array1 = new int[]{2,3,5,7,11,13,17,19};
        int [] array2;

        //显示array1的内容
        for (int i = 0; i < array1.length; i++){
            System.out.print(array1[i] + " ");

        }
        System.out.println();
        //赋值array2变量等于array1 ，修改array2中的偶数索引元素，使其等于索引值。
        //不能称作数组的复制
        array2 = array1;

        for (int i = 0; i < array2.length; i++) {
            if (i % 2 == 0){
                array2[i] = i;

            }

        }
        //显示array1的内容
        for (int i = 0; i < array1.length; i++){
            System.out.print(array1[i] + " ");

        }


    }
}
```
### 输出结果为
```java

2 3 5 7 11 13 17 19 
0 3 2 7 4 13 6 19 
Process finished with exit code 0
```
在例题中的
```java
  //赋值array2变量等于array1 ，修改array2中的偶数索引元素，使其等于索引值。
        //不能称作数组的复制
        
        array2 = array1;
```
这个只是地址的复制，相当于array2 是array1 的快捷方式，并没用新的数组生成。
**新数组的生成必须有new关键字。** 

## 数组的完全复制需要用到下面的方法
```java
使array2复制array1
        array2 = new int[array1.length];
        for (int i = 0; i < array2.length; i++) {
            array2[i] = array1[i];
        }

        for (int i = 0; i < array2.length; i++) {
            if (i % 2 == 0){
                array2[i] = i;

            }

        }
```
输出结果为：
```java

2 3 5 7 11 13 17 19 
2 3 5 7 11 13 17 19

```
